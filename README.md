# FRESH-CARE 🌊❄️
**Unraveling FRESHwater and ocean Currents changes in the Arctic using REmote sensing**

Welcome to the official repository of the **FRESH-CARE** project, an **ERC Starting Grant** hosted by the **Barcelona Polar Lab**.

* **Principal Investigator:** Marta Umbert Ceresuela 📧 mumbert@icm.csic.es
* **Host Institution:** Barcelona-Polar-Lab, Institut de Ciències del Mar (ICM-CSIC), Barcelona
* **Funding:** European Research Council (ERC) Starting Grant

---

## 📂 Repository Structure

This repository acts as a centralized "monorepo" containing all the processing pipelines, machine learning architectures, and modeling configurations developed throughout the project, organized strictly by Work Packages (WPs) and field activities.

```text
fresh-care/
├── WP1/     # Ocean surface currents evaluation
├── WP2/     # LSTM to reconstruct 4D oceanographic data
├── WP3/     # Arctic freshwater content (FWC) analysis
├── WP4/     # Freshwater transport (FWT) across Arctic gateways
├── WP5/     # Arctic Ocean Numerical Model (ROMS-Sea ice)
├── campaigns/  # In-situ data processing and analysis
└── README.md   # This file
