# Field Campaigns & In-Situ Data Processing 🚢

This directory centralizes the processing pipelines, quality control protocols, and post-processing scripts for in-situ datasets collected during oceanographic field campaigns or retrieved from international mooring networks. These datasets provide the baseline validation data for the **FRESH-CARE** project frameworks (WP1, WP2, and WP4).
